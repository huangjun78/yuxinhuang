# 图片资源说明

请将以下图片放置在此目录中：

## 需要的图片文件

1. **hero.jpg** (推荐尺寸: 1200x800)
   - 用途：首页英雄区背景图片
   - 描述：年轻女钢琴家 Yuxin Huang 在音乐厅演奏三角钢琴的场景，优雅的姿势，深色背景，戏剧性灯光，专业音乐摄影

2. **about.jpg** (推荐尺寸: 500x400)
   - 用途：关于部分的肖像图片
   - 描述：年轻女钢琴家 Yuxin Huang 的特写肖像，优雅的演出服装，演奏三角钢琴，柔和灯光，专业摄影

## 图片格式建议

- 格式：JPG 或 PNG
- 优化：建议压缩图片以提升加载速度
- 命名：使用小写字母和连字符，如 `hero.jpg`, `about.jpg`

## 使用方式

图片在代码中通过 `/images/文件名` 的方式引用，例如：
- `/images/hero.jpg`
- `/images/about.jpg`

这些路径会自动指向 `public/images/` 目录。
